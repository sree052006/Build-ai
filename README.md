# Project Name
Guardian AI

## Problem Statement
Elderly individuals living alone face life-threatening risks from falls and medical emergencies that often go undetected for hours. Conventional alert systems require the person to press a button — which is impossible when they are unconscious, disoriented, or physically unable to respond.
There is an urgent need for a fully passive, always-on safety system that can:

Automatically detect falls from CCTV footage without any human trigger
Assess the elder's emotional and cognitive state through voice
Notify caretakers and family members instantly across multiple devices
Track patterns over time and generate intelligent weekly health reports

## Project Description
Guardian AI is a real-time elderly monitoring platform with five interconnected modules accessible from a unified interface:
 * Edge Device (Home):The primary monitoring unit — captures CCTV frames and runs fall detection
 * Dispatch Dashboard:Central alert hub showing live fall events with AI-generated descriptions and "Mark as Seen" workflow
 * Digital Twin: AI-generated behavioral model of the elder — tracks movement patterns, logs historical fall events, detects anomalies, and produces weekly caregiver reports
 *  Caretaker Phone:Instant push notifications and alerts sent to the assigned caretaker
 *  Family Phone:Parallel notifications sent to family members for awareness

   Core Features
1.  Real-Time Fall Detection

Continuously extracts JPEG frames from CCTV video
Sends frames to Gemini 2.5 Flash for pose analysis
AI generates a natural language description of the posture and assigns a severity level (1–10)
If level ≥ 8 → emergency alert is automatically dispatched to Dispatch Dashboard, Caretaker Phone, and Family Phone

2.  Vocal Sentiment Engine

AI proactively asks the elder "Are you okay?"
Transcribed response is analyzed for emotional distress, confusion, or cognitive decline
Response latency is factored in as an additional wellness signal
If distress is detected → dispatch help is triggered

3.  Mini Digital Twin — Movement Pattern Tracking

Builds a behavioral baseline of the elder's daily movement patterns
Tracks and displays:

Falls This Week — total confirmed fall events
Near-Falls — close-call incidents
Anomalies — unusual behavior deviations
Avg Severity — average severity score across all events (e.g., 9.0/10)


Detects pattern anomalies such as:

HIGH FREQUENCY — multiple falls within 24 hours (elevated risk)
WORSENING TREND — severity of events increasing over time


Maintains a Historical Fall Log with timestamped AI descriptions for every event

4.  AI Weekly Caregiver Report

Automatically generated by Gemini at the end of each week
Summarizes fall frequency, severity trends, peak incident times, and anomalies
Provides actionable recommendations (e.g., "Immediate medical review recommended")
Shareable with doctors, family, and care facilities

5.  Multi-Device Alert System

Simultaneous alerts dispatched to:

Dispatch Dashboard (with full AI description + Mark as Seen)
Caretaker's phone
Family members' phones
## Google AI Usage
*Gemini 2.5 Flash (gemini-2.5-flash) — Google DeepMind's multimodal LLM; the sole ML model powering all AI features in Guardian AI

*Task 1 — Computer Vision: Fall Detection
Input:JPEG image frame extracted from CCTV video
Model behavior:Object detection + pose estimation + scene understanding
What it does:Classifies the person's posture, generates a natural language description, and assigns a severity level
Trigger:level ≥ 8 → emergency alert dispatched across all devices

Task 2 — NLP: Vocal Wellness Check
Input:Transcribed speech text + response latency (milliseconds)
Model behavior:Emotion classification + cognitive state analysis
What it does:Detects if the elder sounds confused, distressed, or calm
Trigger:Distress or high latency flag → dispatch help

Gemini 2.5 Flash 
Multimodal:Handles both image AND text in a single API call
Flash variant:Optimized for speed — low latency, critical for real-time monitoring
Free tier:15 requests/min — sufficient for hackathon demo
No training needed:Zero-shot prompting — no dataset collection required


### Tools / Models Used
- *Gemini 2.5 Flash (gemini-2.5-flash) — Google DeepMind's multimodal LLM; the sole ML model powering all AI features in Guardian AI
*Vision (Multimodal)	Analyzes CCTV video frames every 8 seconds, classifies posture as fallen / sitting / sleeping / walking and scores distress 1–10
*Language (NLP Sentiment)	Analyzes the elder's verbal response to "Are you okay?" — detects Confused / Distressed / Calm & Clear / Possibly Declining
*Gemini API (REST)	Called directly from the browser via fetch — no SDK needed

### How Google AI Was Used

AI-generated weekly caregiver report in Digital Twin view
